The Development

Here you can read about the development and what it took to make this game happen.

Who we are:

This game was developed as a student project in the bachelor program of computer science at the University of Tübingen. The developers are: Stephan Amann, Cornelius Bopp, David Kleindiek, Tanja Huber, Tina Truong and Amelie Schäfer.

What we did:

  • Quest System
  • Inventory System
  • NPC AI-Behaviour
  • UI
  • Landscape Design
  • Logo Design
  • Icons
  • Particles
  • Lighting
  • Skill System
  • Movement and Combat System
